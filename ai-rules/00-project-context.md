# Project Context

## What This Project Is
- Salon appointment booking and management system
- React + TypeScript + Tailwind + shadcn/ui
- "Luxe Gold & Black" theme with gold (#d4a746) primary

## Non-Goals
- No backend code (frontend only, mock API)
- No native mobile apps
- No e-commerce/payment processing

## Key Patterns
- Mock data in `src/data/mockData.ts`
- Services layer in `src/services/api.ts`
- Types in `src/types/index.ts`
- Role-based routing: CLIENT and ADMIN
